## UX 디자인이 뭘까?
UX를 디자인하는 것 -> UX 디자인<br>
디자인 : 주어진 목적을 조형적으로 실체화하는 것<br>
UX : User eXperience 사용자 경험<br>
UX : 시스템 / 제품 / 서비스를 이용하면서 느끼고 생각하게 되는 총체적 경험 (사용자가 느끼는 방식)
<br><strong> UX라는 단어는 정확히 이것이다라고 정의하기 어려움. </strong>
